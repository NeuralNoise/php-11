DigiOz Guestbook Help Documentation


Welcome to the DigiOz Guestbook , and thank you for choosing and downloading this FREE Guestbook System. The DigiOz Guestbook is a PHP driven Guestbook system that stores all the entries in a simple text file. It is very simple to install and use, and the theme files included with the script allow you to customize the script to blend in with the rest of your website without knowing any programming.


The guestbook contains features such as entry flooding protection, administrative entry notification, bad word filter, emoticons, and much much more. Current Language support includes English, German, Swedish, Greek, Dutch, Philippino and more (features).



For a complete list of features for any version of the Guestbook visit the following forum thread:
